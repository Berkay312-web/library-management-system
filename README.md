# Library Management System

This project is a terminal-based Library Management System developed in Python.
It is designed to manage books, patrons, and basic circulation operations.

## Setup
- Python 3.x must be installed on the system.
- The project uses a `data/` directory to store runtime JSON files
  (`books.json`, `patrons.json`, `loans.json`).

## How to Run
Open a terminal in the project directory and run:

```bash
python main.py
